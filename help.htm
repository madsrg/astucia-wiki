Wiki Help & Shortcuts

This is a simple file-based wiki. All your pages are stored as Markdown files in the `pages` directory on the server.

Features

  • File/Folder Management: Create, rename, delete, and move pages and folders directly from the sidebar.
  • Drag & Drop: Move files into folders by dragging them.
  • Markdown Editing: Write content using standard Markdown syntax.

Keyboard Shortcuts (in Edit Mode)

  • Save Page: Ctrl + S
  • Heading 1: Ctrl + 1
  • Heading 2: Ctrl + 2
  • Heading 3: Ctrl + 3
  • Bold: Ctrl + B
  • Italic: Ctrl + I
  • Link: Ctrl + K
  • Code: Ctrl + `
